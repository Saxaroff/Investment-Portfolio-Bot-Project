# Математическая модель для AI-оценки портфелей

Исполнитель: DataLead (руководитель Отдела данных)  

## Основные формулы

1. Модель Марковица (Modern Portfolio Theory):
   Ожидаемая доходность портфеля:
   E(R_p) = Σ w_i * E(R_i)

   Риск (стандартное отклонение):
   σ_p = √(Σ Σ w_i * w_j * Cov(R_i, R_j))

2. Value at Risk (VaR):
   VaR_α = - (μ_p - z_α * σ_p)  (где z_α — квантиль нормального распределения)

3. Sharpe Ratio:
   SR = (E(R_p) - R_f) / σ_p

## Реализация в AI
Модель будет использована как loss-функция при обучении нейросети для максимизации Sharpe Ratio.
